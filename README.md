## 🚀 Project Overview
# Face Detection Attendance System

This project implements an automated attendance system using face detection with OpenCV.

## Technologies Used
- Python
- OpenCV
- Computer Vision

## Description
The system captures video using a webcam and detects faces in real time. When a face is detected, attendance is automatically recorded along with a timestamp.

This project demonstrates the application of computer vision techniques in real-world scenarios such as automated attendance systems.

## Features
- Real-time face detection using OpenCV
- Automatic attendance recording
- Timestamp-based tracking
- Simple and efficient implementation
